## 项目经历

### YieldMax - DeFi 收益聚合协议 | 智能合约主程 & 后端架构师
*2022.11 – 至今*

- **项目描述：** 一个自动在不同 DeFi 协议间进行资金迁移以优化用户收益的聚合器。TVL 最高达 500 万美元。
- **核心技术：** Solidity, Hardhat, NestJS, TypeScript, The Graph, Chainlink
- **主要职责与成果：**
    -   使用 Solidity 和 Hardhat 框架开发了核心 Vault 和策略合约，采用检查-效果-交互模式严防重入攻击。
    -   集成 Chainlink 预言机获取安全的资产价格，并实现基于 ERC-4626 标准的代币化份额。
    -   使用 NestJS 构建后端服务，监控链上事件、计算实时收益率并执行自动化复合策略。
    -   开发离线签名中继服务，支持用户无 Gas 存款；使用 The Graph 创建 Subgraph 为前端提供高效数据查询。
    -   **成果：** 协议安全稳定运行超 6 个月，累计为用户创造收益超 50 万美元，并通过第三方安全审计。

### MetaCanvas - NFT 铸造与交易平台 | 全栈开发工程师
*2022.03 – 2022.10*

- **项目描述：** 一个支持生成艺术 NFT 一键铸造、二级市场交易和版税分成的平台。
- **核心技术：** Solidity, ERC-721A, Express.js, IPFS, Moonpay API
- **主要职责与成果：**
    -   基于 ERC-721A 标准开发 Gas 优化的 NFT 合约，支持白名单和分批铸造。
    -   开发版税合规的市场合约（遵循 EIP-2981），实现自动创作者分成。
    -   构建链下元数据生成器并上传至 IPFS，使用 Express.js 开发平台后端 API。
    -   集成 Moonpay，支持用户使用法币直接购买 NFT。
    -   **成果：** 成功上线 3 个 NFT 系列，总交易量超过 2000 ETH。
